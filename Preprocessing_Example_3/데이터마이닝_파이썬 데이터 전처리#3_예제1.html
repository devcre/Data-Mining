1. 데이터 분리¶

데이터 분리는 크게 다음과 같이 두 단계로 구성됨

(1) feature와 label의 분리

  • feature와 label의 분리는 별도로 X와 Y로 별도로 정의해줘도 되고, 데이터 프레임의 슬라이싱을 통해 인덱싱해도 됨

(2) 학습 데이터와 평가 데이터의 분리

  • 학습 데이터와 평가 데이터의 분리에는 train_test_split 함수를 사용함
  • train_test_split함수는 입력으로 X(학습 데이터), Y(평가 데이터), test_size(평가 데이터의 비율)을 갖음
  • test_size를 입력하지 않은 경우에는 기본 값인 0.25로 설정됨

2. 모델 선택 ~ 3. 모델 인스턴스화 및 파라미터 설정¶

모델 선택은 지도학습에 어떠한 모델을 선택할 것인지를 결정하는 단계로 분석 목적에 맞는 모델을 선택하는 것이 중요함

본 예제에서는 Decision Tree 모델 적용 방법을 소개함

< Decision Tree 모델 >¶

  • Decision Tree는 sklearn.tree 모듈의 DecisionTreeClassifier 함수를 활용함

  • Decision Tree의 주요 파라미터

      - criterion : 분류 규칙에 대한 파라미터로, 두 가지 설정이 있음(기본값은 "gini")
        - "gini" for the Gini impurity and "entropy" for the information gain.
        - 예) tree.DecisionTreeClassifier(criterion="entropy")
      - max_depth : 나무의 최대 깊이에 대한 파라미터로, 정수 값을 입력으로 받음(기본값은 'None')
      - min_samples_split : 노드 당 최소 샘플 수에 대한 파라미터로, 정수 값을 입력으로 받음(기본값은 2)
  • 다른 파라미터들에 대한 내용: https://scikit-learn.org/stable/modules/generated/sklearn.tree.DecisionTreeClassifier.html

6. 모델 평가¶

모델 적용 결과에 대해 실제 값(test_y)과 예측 값(predicted)을 비교하여 평가하는 단계를 말함

  • scikitlearn.metrics 모듈에서 제공하는 주요 모델 평가 함수

      - confusion_matrix(test_y, predicted)
      - classification_report(test_y, predicted)
      - 분류 성능 평가 지표
        - 정확도: accuracy_score(test_y, predicted)
        - 정밀도: precision_score(test_y, predicted)
        - 재현율: recall_score(test_y, predicted)
        - F1점수: f1_score(test_y, predicted)

  • 자세한 내용은 https://scikit-learn.org/stable/modules/model_evaluation.html#classification-metrics 에서 확인할 수 있음

## 7. 결과 해석 및 시각화
import io
import pydot
from IPython.core.display import Image
from sklearn.tree import export_graphviz

def draw_decision_tree(model):
    dot_buf = io.StringIO()
    export_graphviz(model, out_file=dot_buf, feature_names=Train_X.columns, 
                    class_names=Train_y.unique(), filled=True)
    graph = pydot.graph_from_dot_data(dot_buf.getvalue())[0]
    image = graph.create_png()
    return Image(image)

draw_decision_tree(model_DT)
