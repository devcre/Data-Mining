2. 모델 선택 ~ 3. 모델 인스턴스화 및 파라미터 설정¶

모델 선택은 지도학습에 어떠한 모델을 선택할 것인지를 결정하는 단계로 분석 목적에 맞는 모델을 선택하는 것이 중요함

본 예제에서는 k-NN 모델 적용 방법을 소개함

< k-Nearest Neighbors 모델 >¶

  • kNN은 sklearn.neighbors 모듈의 KNeighborsClassifier 함수를 활용함
  • kNN의 주요 파라미터
      - n_neighbors : 최소 이웃의 수(기본 값은 5)
      - metric: 거리 척도에 대한 파라미터로, 기본값은 민코우스키 거리("minkowski")임
          유클리드안 거리("euclidean"), 맨하탄 거리("manhattan"), 마할라노비스 거리("mahalanobis")
  • 다른 파라미터에 대한 자세한 내용: https://scikit-learn.org/stable/modules/generated/sklearn.neighbors.KNeighborsClassifier.html

6. 모델 평가¶

모델 적용 결과에 대해 실제 값(test_y)과 예측 값(predicted)을 비교하여 평가하는 단계를 말함

  • scikitlearn.metrics 모듈에서 제공하는 주요 모델 평가 함수

      - confusion_matrix(test_y, predicted)
      - classification_report(test_y, predicted)
      - 분류 성능 평가 지표
        - 정확도: accuracy_score(test_y, predicted)
        - 정밀도: precision_score(test_y, predicted)
        - 재현율: recall_score(test_y, predicted)
        - F1점수: f1_score(test_y, predicted)

  • 자세한 내용은 https://scikit-learn.org/stable/modules/model_evaluation.html#classification-metrics 에서 확인할 수 있음

##추가. Best K 찾기(정확도 기준)
training_accuracy = []
test_accuracy = []

# 3에서 13까지 n_neighbors를 적용
neighbors_range = range(5, 14, 2) # 짝수인 k값은 일반적으로 잘 사용하지 않음

for n_neighbors in neighbors_range:
    kNN = KNeighborsClassifier(n_neighbors=n_neighbors)
    kNN.fit(Train_X, Train_y)
    training_accuracy.append(kNN.score(Train_X, Train_y)) # 학습 데이터에 대한 분류 정확도
    test_accuracy.append(kNN.score(Test_X, Test_y)) # 평가 데이터에 대한 분류 정확도
plt.rcParams['figure.figsize'] = [15, 8]
plt.plot(neighbors_range, training_accuracy, label="Training accuracy")
plt.plot(neighbors_range, test_accuracy, label="Test accuracy")
plt.ylabel("Accuracy")
plt.xlabel("k value")
plt.text(np.arange(5, 14, 2)[training_accuracy== max(training_accuracy)]-0.2,
         max(training_accuracy)+0.0004, "Best at k=9")
plt.legend()
plt.show()

#추가. kNN 모델 평가 결과(2차원 시각화)
from sklearn import preprocessing
from matplotlib.colors import ListedColormap

le = preprocessing.LabelEncoder()
le.fit(Train_y.unique())

Train_X_2d = np.array(Train_X)[:, [1,3]] # 3차원은 시각화하기 어렵기 때문에 2차원으로 특징을 임의로 선택했음
Train_y_to_int = le.transform(Train_y) # iris의 label을 각각 0, 1, 2로 변환

Test_X_2d = np.array(Test_X)[:, [1,3]]
Test_y_to_int = le.transform(Test_y)
h = .02  # step size in the mesh

# Create color maps
cmap_light = ListedColormap(['#FFAAAA', '#AAFFAA', '#AAAAFF'])
cmap_bold = ListedColormap(['#FF0000', '#00FF00', '#0000FF'])

for n_neighbor in [1, 3, 5]:
    knn = KNeighborsClassifier(n_neighbor)
    knn.fit(Train_X_2d, Train_y_to_int)

    # Plot the decision boundary. For that, we will assign a color to each
    # point in the mesh [x_min, x_max]x[y_min, y_max].
    x_min, x_max = Train_X_2d[:, 0].min() - 1, Train_X_2d[:, 0].max() + 1
    y_min, y_max = Train_X_2d[:, 1].min() - 1, Train_X_2d[:, 1].max() + 1
    xx, yy = np.meshgrid(np.arange(x_min, x_max, h),
                         np.arange(y_min, y_max, h))
    Z = knn.predict(np.c_[xx.ravel(), yy.ravel()])

    # Put the result into a color plot
    Z = Z.reshape(xx.shape)
    plt.figure()
    plt.pcolormesh(xx, yy, Z, cmap=cmap_light)

    # Plot also the training points
    plt.scatter(Test_X_2d[:, 0], Test_X_2d[:, 1], c=Test_y_to_int, cmap=cmap_bold,
                edgecolor='k', s=20)
    plt.xlim(xx.min(), xx.max())
    plt.ylim(yy.min(), yy.max())
    plt.xlabel(Train_X.columns[1])
    plt.ylabel(Train_X.columns[3])
    plt.title("3-Class classification (k = %i)"
              % (n_neighbor))

plt.show()
